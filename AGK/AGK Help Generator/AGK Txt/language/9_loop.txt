loop

_sheader
Syntax

_tier1
loop
_tier1

_sheader
Description

_text
These commands will define a loop that will repeat indefinitely. You are able to break from the loop at any time by using the <i>exit</i> command.

_sheader
Example

_tier1
do
    if GetPointerPressed ( ) = 1
        exit
    endif
loop
