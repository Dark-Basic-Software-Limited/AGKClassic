App Store testing

In order to test your application on the App Store you will need to upload it using the Application Loader program. Run Application Loader (you can find it by using the search feature in Mac OS and typing in Application Loader), select Deliver Your App and follow the on screen prompts.

Your application will be automatically processed by Apple. Once this processing has been completed and your application is ready for testing you will be notified by email. At this point you can invite users to test your application.

  • Login to https://itunesconnect.apple.com/
  • Click on My Apps
  • Click on the application you want to work with
  • Select TestFlight
  • Select iOS under TestFlight builds
  • Add your build to TestFlight
  • Add any emails to the list of testers

Once you have filled in the relevant sections any users in the list will be emailed with instructions on how to install the application.
